# Blinkit Sales Dashboard

## Project Overview
This Power BI dashboard analyzes Blinkit grocery sales data and provides insights into sales performance, outlet performance, item categories, and customer ratings.

## KPIs
- Total Sales
- Average Sales
- Number of Items
- Average Rating

## Features
- Interactive filters
- Sales trend analysis
- Outlet performance analysis
- Item category analysis

## Tools Used
- Power BI
- Power Query
- DAX
- Data Modeling

## Key Insights
- Tier 3 outlets generated the highest sales.
- Regular fat products contributed significantly to revenue.
- Fruits and Snacks categories performed strongly.
